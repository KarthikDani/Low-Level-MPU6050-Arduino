uint8_t i2c_read_byte ( bool  ack)

Reads a single byte from the I2C bus.

This function reads a byte from the slave device and then sends an acknowledgment (ACK) or non-acknowledgment (NACK) based on the ack parameter.

Parameters
ackA boolean flag to determine whether to send an ACK (true) or a NACK (false) after reading the byte.
Returns
The 8-bit data byte read from the bus.

◆ i2c_sda_high()

void i2c_sda_high ( )

Sets the SDA line high by configuring the pin as an input with the internal pull-up disabled.

I2C requires the bus lines to be open-drain, so setting a pin as an input effectively "releases" it, allowing an external pull-up resistor to pull it high.

bool i2c_write_byte ( uint8_t  data)

Writes a single byte over the I2C bus.

This function sends a single byte of data, most significant bit (MSB) first, and then waits for an acknowledgment (ACK) from the slave device.

Parameters
dataThe byte to write.
Returns
true if an ACK is received (SDA is pulled low by the slave), false otherwise (NACK).
